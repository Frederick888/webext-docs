==========================
Changes in Thunderbird 101
==========================

cloudfile API
=============
* Support for manifest key ``data_format`` has been removed, the only supported format now is ``File``.

messageDisplay API
==================
* :ref:`messageDisplay.getDisplayedMessage` now returns messages in inactive tabs.
